# Test Suite

This directory contains the comprehensive test suite for the project.

## Structure
- `unit/`: Unit tests (C/C++).
- `integration/`: Integration tests.
- `functional/`: Functional/E2E tests (Python/Shell).
- `performance/`: Performance benchmarks.
- `stress/`: Stress tests.
- `fixtures/`: Test data.

## Running Tests
See `Makefile` or root `README.md` for instructions.
